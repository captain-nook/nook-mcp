# Nook MCP

统一存放 nook 的 MCP servers。

## Servers

| Directory | Description | Runtime |
|-----------|-------------|---------|
| `nook-image-gpt` | GPT Image 2.0 generation via relay API | Node.js >= 18 |

## Quick Start

```bash
cd <server-dir>
# See each directory's README.md for setup instructions
```

Each server directory is self-contained with its own `setup.js` for guided configuration.
